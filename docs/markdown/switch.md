# Switch 开关组件

---
Switch 开关组件

以下文档指示例子，请根据具体组件编写

## 使用指南

在 Taro 文件中引入组件

:::demo

```js
import AtSwitch from 'taro-ui'
```

:::

## 一般用法

:::demo

```html
<AtForm>
  <AtSwitch title='开启中' checked onChange={val => console.log(val)} />
  <AtSwitch title='已关闭' />
</AtForm>


```

:::

## 参数

| 参数       | 说明                                   | 类型    | 可选值                                                              | 默认值   |
| ---------- | -------------------------------------- | ------- | ------------------------------------------------------------------- | -------- |
| title | 标签名 | String  | - | - |
| color | 背景颜色  | String  | - | `#6190e8` |
| checked     | 是否显示开启  | Boolean | - | false |


## 事件

| 事件名称 | 说明          | 返回参数  |
|---------- |-------------- |---------- |
| onChange | 输入框值改变时触发的事件 | 当前值value  |
