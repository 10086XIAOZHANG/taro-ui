# FloatLayout 浮动布局

---

## 使用指南

在 Taro 文件中引入组件

:::demo

```js
import { AtFloatLayout } from "taro-ui"
```

:::

## 一般用法

:::demo

```html
<AtFloatLayout></AtFloatLayout>
```

:::

## AtFloatLayout 参数

| 参数     | 说明                 | 类型    | 可选值 | 默认值  |
| -------- | -------------------- | ------- | ------ | ------- |
| title    | 元素的标题           | String  | -      | -       |
| isOpened | 控制是否出现在页面上 | Boolean | -      | `false` |
