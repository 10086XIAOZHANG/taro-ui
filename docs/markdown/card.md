# Card 浮动布局

---

## 使用指南

在 Taro 文件中引入组件

:::demo

```js
import { AtCard } from "taro-ui"
```

:::

## 一般用法

:::demo

```html
<AtCard></AtCard>
```

:::

## AtCard 参数

| 参数    | 说明                 | 类型     | 可选值 | 默认值 |
| ------- | -------------------- | -------- | ------ | ------ |
| title   | 元素的标题           | String   | -      | -      |
| note    | 元素的辅助信息       | String   | -      | -      |
| thumb   | 元素的缩略图         | String   | -      | -      |
| extra   | 元素的额外信息       | String   | -      | -      |
| onClick | 元素被点击触发的事件 | Function | -      | -      |
